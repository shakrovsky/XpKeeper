## Вступление пользователя в чат  
<img width="760" height="299" alt="new_member" src="https://github.com/user-attachments/assets/fb930dd9-8161-4f47-9c03-2f6c0e2e36ca" />  
<img width="760" height="299" alt="old_member" src="https://github.com/user-attachments/assets/ab489176-ee0b-4d66-ba0b-8ee837b5ccdb" />  

- При первом вступлении в чат бот отправляет приветственное сообщение и автоматически регистрирует пользователя в системе.  
- После регистрации создаётся профиль, который сохраняется навсегда.  
- Если пользователь покидает чат и возвращается снова, бот распознаёт его и встречает другим сообщением, подтверждая, что профиль уже существует, а все данные (XP, титулы и т.д.) сохранены.

## Команда /me  
<img width="435" height="275" alt="__me" src="https://github.com/user-attachments/assets/2772f5c8-86cf-46fa-8761-df08a49f4ec9" />  

Команда показывает:  
- текущий титул пользователя;  
- его уровень XP.  

Титулы бывают двух видов:  
- **базовые** — автоматически присваиваются в зависимости от уровня XP;  
- **уникальные** — назначаются индивидуально.  

Под «титулом» понимается персональное звание, которое может быть закреплено за каждым администратором чата.  

P.S. Каждый участник чата (кроме приближённых) получает статус администратора с минимальным набором прав.  

[‹ Обратно к оглавлению](#xpkeeper)

## Команда /top  
<img width="366" height="377" alt="__top" src="https://github.com/user-attachments/assets/2641e9fc-f879-4db9-a7df-2cd13f2c3731" />  

Команда выводит рейтинг 20 пользователей с наибольшим количеством XP (в порядке убывания).  
В список попадают только пользователи с зарегистрированными профилями.  

Особенности:  
- имена в списке кликабельны и ведут на профиль пользователя;  
- данные (имя и количество XP) отображаются актуальными на момент вызова команды.

## Команда /titles  
<img width="489" height="340" alt="__titles" src="https://github.com/user-attachments/assets/d58910fa-c7c4-4a45-a414-31dec033554c" />  
<img width="575" height="225" alt="__titles_after_edit" src="https://github.com/user-attachments/assets/3afcbec0-e837-4749-b119-cc453215092c" />  
<img width="699" height="537" alt="__alert_true" src="https://github.com/user-attachments/assets/0a56312a-46d5-4d39-9b15-f45d5aac9277" />  

Команда выводит текущий титул пользователя (тот, который установлен в Telegram).  

К сообщению прикрепляется inline-клавиатура. Каждая кнопка на ней соответствует одному из титулов, доступных пользователю (см. фото 1).  

При выборе титула:  
- исходное сообщение редактируется и отображает новый титул;  
- титул пользователя в Telegram обновляется автоматически;  
- inline-клавиатура исчезает (см. фото 2, пример с кнопкой «Киборг»).  

Особенности использования:  
- команду можно безопасно вызывать в общем чате;  
- попытки других пользователей (включая доверенных) изменить чужой титул игнорируются: им отображается уведомление об ошибке (см. фото 3).
