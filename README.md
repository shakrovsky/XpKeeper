## Вступление пользователя в чат:

<img width="760" height="299" alt="old_member" src="https://github.com/user-attachments/assets/ab489176-ee0b-4d66-ba0b-8ee837b5ccdb" />

Когда пользователь вступает в чат впервые, бот выводит приветственное сообщение и автоматически регистрирует его аккаунт в системе.

Это означает, что если пользователь выйдет из чата и затем снова вступит в него, вся его информация (XP, титулы и прочее) сохранится.

Если же пользователь вступает в чат повторно, бот встречает его другим сообщением, с упоминанием того, что он уже известен системе и его данные сохранены.
[Обратно к оглавлению](#xpkeeper)

## Команда /me:

Команда показывает текущий титул пользователя и уровень его XP. 
Под «титулом» понимается кастомная должность, которую можно назначить каждому администратору чата.
